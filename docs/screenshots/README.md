# Captures d’écran pour le README

Ajoutez ici 1 à 2 captures pour la section **Aperçu produit** du README racine.

**Recommandé :**

- `dashboard.png` – écran Dashboard (KPIs, graphiques).
- `list.png` – une liste (ex. Comptes ou Transactions).

**Format :** PNG ou WebP, largeur raisonnable (ex. 800–1200 px) pour une bonne lisibilité sur GitHub.

Une fois les fichiers ajoutés, décommentez les lignes correspondantes dans le [README principal](../../README.md) (section « Aperçu produit »).
